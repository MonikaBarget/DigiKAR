# Geohumanities Tutorials

In einem separaten Github-Repository teilt Monika Barget [Code und Tutorials](https://monikabarget.github.io/GeoHumTutorials/) zur Geokodierung historischer räumlicher Daten mit den Google- und Geonames-APIs. Einige der Skripte enthalten auch integrierte Mapping-Funktionalitäten, um testweise statische oder interaktive Visualisierungen zu erstellen.

# Bearbeitung von Ortsdaten

1. [Geocodierung mit OpenCage API und Plotting auf statischer Karte](https://github.com/ieg-dhr/DigiKAR/JupyterNotebooks_DigiKAR/Geocode-Plot_OpenCage.ipynb)
2. [Geocodierung mit Geonames API und Plotting auf statischer Karte](https://github.com/ieg-dhr/DigiKAR/JupyterNotebooks_DigiKAR/Geocode-Plot_Geonames.ipynb)
3. [Geocodierung mit Geonames API und Plotting auf interaktiver Karte](https://github.com/ieg-dhr/DigiKAR/JupyterNotebooks_DigiKAR/Geocode-Plot_Geonames_interactiveMAP.ipynb)
4. [Geocodierung mit Google API und Plotting auf interaktiver Karte](https://github.com/ieg-dhr/DigiKAR/JupyterNotebooks_DigiKAR/Geocode_Plot_GoogleAPI_interactiveMAP.ipynb)
5. [Geocodierung mit Google API und Plotting auf interaktiver Karte (Daten aus AP3)](https://github.com/ieg-dhr/DigiKAR/JupyterNotebooks_DigiKAR/Geocode_Plot_GoogleAPI_interactiveMAP_AP3.ipynb)
